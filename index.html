Windows桌面应用

安卓桌面 (ADS)

一个运行在Windows环境下的安卓风格桌面应用，模拟安卓手机/平板的桌面体验。

主要功能

  • 安卓风格的桌面界面，包括应用图标网格和底部导航栏
  • 支持自定义主题颜色和背景类型
  • 实时显示时间和状态栏信息
  • 设置面板，可配置主题颜色、背景和自启动选项
  • 系统托盘图标，支持最小化到托盘和快速退出
  • 支持自定义添加软件到桌面

技术栈

HTML5 CSS3 JavaScript Node.js

文件路径

c:\我自制的软件。\ADS

使用说明

快速启动：双击运行 run.bat 文件，首次启动时会自动安装必要的依赖。

控制中心：点击屏幕顶部的任意位置可以打开/关闭控制中心。

自定义软件：通过设置界面可以添加自定义软件到桌面。

退出全屏：按Win键可以退出全屏，访问Windows任务视图。

智能助手应用

PhoneAI

基于Node.js和Electron开发的智能助手应用，支持语音交互、多平台适配和悬浮窗口设计。

主要功能

  • 智能语音交互，无需手动输入
  • 多平台支持,已支持Windows,macOS,Linux没被支持。
  • 编译后生成Android风格界面，包含WiFi、电池、应用图标等功能模拟
  • 悬浮窗口设计，随时可以访问
  • 支持OpenAI和硅基流动等多种AI服务
  • 个性化设置，自定义界面样式和API配置

技术栈

Node.js Electron HTML5 CSS3 JavaScript AI API

文件路径

c:\我自制的软件。\phoneai-node\dist\PhoneAI-win32-x64

使用说明

启动应用：运行PhoneAI.exe可执行文件。

交互方式：支持文字和语音输入与AI助手交流。

悬浮窗口：左键点击打开交互窗口，右键点击显示快捷菜单。

系统托盘：应用最小化时会显示在系统托盘。

API配置：在设置界面可以配置不同AI提供商的API密钥。
